### 根因分析
OpenClaw 新版本(2026.3.24)的 Docker 部署架构改变：
- 旧版：HTTP API + WebSocket 都支持
- 新版：只支持 WebSocket，需要 ECDSA 签名认证
- 1Panel 部署的 Gateway 没有正确配置 HTTP reverse-proxy
