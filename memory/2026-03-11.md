#### **支持的事件类型：**
```
✅ message.received - 收到新邮件
✅ message.sent     - 发送邮件  
✅ message.delivered - 邮件已送达
✅ message.bounced  - 邮件退回
✅ message.opened   - 邮件被打开
✅ message.clicked  - 链接被点击
✅ domain.verified  - 域名验证
```

#### **内置自动化功能：**
1. **智能分类** - 根据内容自动打标签
2. **自动回复** - 对新邮件发送确认回复
3. **通知系统** - 可集成Slack/Discord/Telegram
4. **日志记录** - 完整的操作审计日志
